Descripción del Sistema

Este sistema permite gestionar un invernadero automatizado. Se cargan configuraciones desde un archivo XML de entrada que define invernaderos, hileras de plantas, drones y planes de riego. A partir de estos datos, el sistema genera:

  • Archivo XML de salida con métricas de riego.
  • Reporte HTML con la información de los invernaderos.
  • Gráficas con Graphviz mostrando el estado de los TDAs en un tiempo dado.

Estructuras de Datos Utilizadas

El proyecto utiliza implementaciones propias de TDAs:

  • Lista Simple Enlazada: para almacenar hileras, drones, planes e instrucciones.
  • Cola (simulada con lista enlazada): para manejar la secuencia de instrucciones de riego.
